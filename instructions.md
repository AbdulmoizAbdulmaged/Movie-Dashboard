
#### Setup and Installation
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/AbdulmoizAbdulmaged/Movie-Dashboard.git
   cd movie-dashboard
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```

5. **Run Tests**:
   ```bash
   npm test
   ```
